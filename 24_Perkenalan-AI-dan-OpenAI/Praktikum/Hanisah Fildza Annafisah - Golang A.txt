2. Word classifier


Sistem : 
Anda akan diberikan tweet, dan tugas Anda adalah mengklasifikasikan sentimennya menjadi positif, netral, atau negatif.


Memungkinkan sistem untuk mengklasifikasikan sentimen dalam berbagai bahasa selain bahasa Inggris


Selain klasifikasi menjadi positif, netral, atau negatif, Anda juga dapat mempertimbangkan untuk mengklasifikasikan sentimen menjadi lebih rinci, seperti sangat positif, positif, netral, negatif, dan sangat negatif.


User :
Aku suka bahasa Golang


Assistant :
Sentimen: Positif


User :
I really like the Golang language.


Assistant :
Sentiment: Very positive


User :
Aku tidak suka tidur


Assistant :
Sentimen: Negatif


User :
I really don't like sleeping


Assistant :
Sentimen: Very negative


User :
Buku Golang


Assistant :
Sentimen: Netral
